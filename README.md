# 🪶 Nosce-Codicem: 네 코드를 알라
*A CLI-based visualization tool for understanding your Python logic.*

---

### 🧩 Motivation
알고리즘 문제를 풀 때 반복문이나 조건문 내부에서 변수와 배열의 값 변화를 직접 추적하고 싶을 때가 많습니다.  
하지만 그때마다 테스트 코드를 새로 작성해야 하는 과정이 번거롭고 비효율적이었습니다.  
이에 착안하여, **코드 한 줄만 실행해도 변수의 변화를 자동으로 추적하고 시각화해주는 도구**를 만들고자 프로젝트를 기획했습니다.

---

### 🚀 Project Goal
- **CLI 기반의 알고리즘 학습 보조 도구**를 개발하여,  
  학습자가 자신의 코드 흐름과 변수 변화를 직관적으로 이해할 수 있도록 돕습니다.  
- 최종적으로 **PyPI에 배포 가능한 오픈소스 파이썬 라이브러리**로 완성하는 것을 목표로 합니다.

---

### ⚙️ 구상 중인 기능
- **변수 추적:** 조건문(`if`), 반복문(`for`, `while`) 내 변수 및 군집 자료형(list, dict 등)의 값 변화 자동 추적  
- **재귀함수 분석:** 재귀 호출 단계별 변수 변화 시각화  
- **콘솔 시각화:** `rich` 등의 라이브러리를 활용해 셸 창에서 색상·표·그래프 형태로 출력  
- **가시적 피드백:** 변수의 자료형에 따라 맞춤형 시각화 제공  
- **(추가 예정)** 실행 시간 및 메모리 사용량 측정 후 시각화
